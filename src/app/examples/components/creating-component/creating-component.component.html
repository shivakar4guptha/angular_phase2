Component
A component controls a patch of screen called a view. To write a component in Angular we have to import dependencies intially using importing(explained in detail below) After importing we have to define Decorator to the component to describe its Structure (Emplained below) Having a class to a component is necessary. Class to component is similar to brain for humans
Creating a component in Angular
There are couple of ways to create a component.
Manual Creation Create html, ts, spec, css files with prescribed extensions and import component clas in module file Using command line Command used to create component   "ng generate component componentName" in short "ng g c compName" Note: Dont give spaces in component name
Component Structure
importing import all the dependencies of the component in the import section. For example: "import { Component, OnInit } from '@angular/core';" Decorator Decorator decides behaviour of the class (Component/Service/Module/Service etc..) Example: @Component({..}) / @Module({..}) / @Injectable({..}) Class Blue print of the com ponent, this contains any number of methods and variables Example: export class ComponentName {..}
